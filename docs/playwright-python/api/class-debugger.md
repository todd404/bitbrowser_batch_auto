# Debugger

API for controlling the Playwright debugger. The debugger allows pausing script execution and inspecting the page. Obtain the debugger instance via [browser_context.debugger](https://playwright.dev/python/docs/api/class-browsercontext#browser-context-debugger).

## Methods

### next

Added in: v1.59

Resumes script execution and pauses again before the next action. Throws if the debugger is not paused.

**Usage**

```python
debugger.next()

```

**Returns**

- [NoneType](https://docs.python.org/3/library/constants.html#None)[#](https://playwright.dev#debugger-next-return)

### request_pause

Added in: v1.59

Configures the debugger to pause before the next action is executed.

Throws if the debugger is already paused. Use [debugger.next()](https://playwright.dev/python/docs/api/class-debugger#debugger-next) or [debugger.run_to()](https://playwright.dev/python/docs/api/class-debugger#debugger-run-to) to step while paused.

Note that [page.pause()](https://playwright.dev/python/docs/api/class-page#page-pause) is equivalent to a "debugger" statement — it pauses execution at the call site immediately. On the contrary, [debugger.request_pause()](https://playwright.dev/python/docs/api/class-debugger#debugger-request-pause) is equivalent to "pause on next statement" — it configures the debugger to pause before the next action is executed.

**Usage**

```python
debugger.request_pause()

```

**Returns**

- [NoneType](https://docs.python.org/3/library/constants.html#None)[#](https://playwright.dev#debugger-request-pause-return)

### resume

Added in: v1.59

Resumes script execution. Throws if the debugger is not paused.

**Usage**

```python
debugger.resume()

```

**Returns**

- [NoneType](https://docs.python.org/3/library/constants.html#None)[#](https://playwright.dev#debugger-resume-return)

### run_to

Added in: v1.59

Resumes script execution and pauses when an action originates from the given source location. Throws if the debugger is not paused.

**Usage**

```python
debugger.run_to(location)

```

**Arguments**

-

`location` [Dict](https://docs.python.org/3/library/typing.html#typing.Dict)[#](https://playwright.dev#debugger-run-to-option-location)

  -

`file` [str](https://docs.python.org/3/library/stdtypes.html#text-sequence-type-str)

  -

`line` [int](https://docs.python.org/3/library/stdtypes.html#numeric-types-int-float-complex) _(optional)_

  -

`column` [int](https://docs.python.org/3/library/stdtypes.html#numeric-types-int-float-complex) _(optional)_

The source location to pause at.

**Returns**

- [NoneType](https://docs.python.org/3/library/constants.html#None)[#](https://playwright.dev#debugger-run-to-return)

Added in: v1.59

## Events

### on("pausedstatechanged")

Added in: v1.59

Emitted when the debugger pauses or resumes.

**Usage**

```python
debugger.on("pausedstatechanged", handler)

```
